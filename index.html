Qué es Libre Código

Programación visual que convierte código real

Libre Código es un IDE de programación visual que convierte de forma bidireccional entre código real y bloques visuales. Escribe código normal y véalo como bloques, o arrastra bloques y genera código ejecutable.

Soporta Python, JavaScript y Batch. Diseñado para principiantes y profesionales por igual.

Programación Visual

Arrastra y conecta bloques para construir la lógica de tu programa de forma intuitiva.

Código Real

Genera código Python, JavaScript o Batch ejecutable a partir de tus bloques visuales.

Multiplataforma

Disponible en Windows, Linux, macOS y directamente desde tu navegador.

Extensible

Sistema de plugins para añadir linters, integraciones y funcionalidades extra.

Por qué se creó

Problemas que todos conocemos

El desarrollo moderno ha acumulado complejidad innecesaria. Libre Código fue creado para resolver estos problemas.

Complejidad innecesaria

Configurar un entorno de desarrollo moderno requiere instalar, configurar y mantener docenas de herramientas antes de escribir una línea de código.

Dependencia de herramientas externas

Los desarrolladores dependen de una cadena interminable de dependencias, gestores de paquetes y configuraciones que rompen constantemente.

Barrera de entrada

Aprender a programar debería ser accesible para todos. La configuración técnica aleja a miles de personas potencialmente brillantes.

Libre Código democratiza el desarrollo de software eliminando la complejidad innecesaria y ofreciendo una experiencia que combina lo mejor de ambos mundos: la potencia del código real y la accesibilidad de la programación visual.

SDK para Plugins

Desarrolla tu propio plugin

SDK completo con API para crear, registrar y distribuir plugins en el ecosistema de Libre Código.

// plugin.json - Estructura básica de un plugin { "name": "mi-plugin", "version": "1.0.0", "description": "Descripción del plugin", "main": "index.js", "apiVersion": "1.0" } // index.js - Registro del plugin import { registerPlugin, addBlock } from '@libre-codigo/sdk'; registerPlugin({ onActivate() { addBlock({ type: 'mi-bloque', label: 'Mi Bloque Personalizado', category: 'custom', execute(ctx) { // Lógica del bloque } }); } });
